# Python-Banking-System
This project is a simple console-based banking system implemented in Python.
It uses object-oriented programming (OOP) principles and Pickle serialization to store customer account data persistently in a file (customerData.pkl).

The system allows users to perform basic banking operations such as creating accounts, checking balances, withdrawing, depositing, and transferring money between accounts.

✨ Features
- Create Account: Generates a new account with a random account number and stores customer details.

- Check Balance: Displays account details including balance.

- Withdraw: Deducts money from an account if sufficient balance exists.

- Deposit: Adds money to an account.

- Transfer: Transfers money between two accounts.

- Persistent Storage: All customer data is stored in a .pkl file using Python’s pickle module.

- Menu-driven Interface: Easy navigation with numbered options.

🛠️ Technologies Used
- Python 3
- Pickle module (for serialization)
- Random module (for generating account numbers)
- Object-Oriented Programming (OOP)
